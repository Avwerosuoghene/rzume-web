## [Unreleased]

### Added
- Initial project setup with Angular 18.2.0
- User authentication with Google OAuth and JWT
- Job application dashboard with CRUD operations
- Responsive design with Angular Material
- State management with RxJS BehaviorSubjects
- Comprehensive testing setup (Jasmine, Karma, Cypress)
- CI/CD pipeline with GitHub Actions
- Docker deployment to Google Cloud Run
- Windsurf configuration with skills, agents, and rules
- Documentation automation with Husky pre-commit hooks

### Changed
- Restructured dashboard template to fix search input clearing bug
- Improved skeleton loader integration

### Fixed
- Search input clearing during job list reload
